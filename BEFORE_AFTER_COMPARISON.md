## New Page: revops.html

### WHAT WAS ADDED
A comprehensive 5-pillar platform page that:

1. **Revenue Capture**
   - Full explanation of voice AI, SMS/email endpoints
   - Shows: Inbound Calls UI with capture status
   - Backed by: RetellAI, Twilio, SendGrid, consent enforcement

2. **Revenue Conversion**
   - Appointment lifecycle from intent to arrival
   - Shows: Workflow visualization with show rate metrics
   - Backed by: Cal.com, scheduling APIs, automated reminders

3. **Revenue Retention**
   - Patient lifetime value and recall automation
   - Shows: LTV trends and campaign tracking
   - Backed by: Patient timeline, marketing campaigns, portal APIs
   - Honest positioning about rollout status

4. **Revenue Intelligence**
   - AI execution vs suggestion differentiation
   - Shows: Healix AI chat interface and agent actions
   - Backed by: Streaming context, safe actions, Inngest, audit logs
   - Marked as "biggest differentiator"

5. **Revenue Control**
   - Complete compliance and audit infrastructure
   - Shows: Compliance badges and recent audit log
   - Backed by: Multi-tenant, PHI redaction, role-based access

**Impact:** Provides deep-dive into each pillar with technical backing and honest positioning.
